# ai-hands-on

## CODE BREAKER — HIT & BLOW

サイバー・ハッカー風の演出付きヒット＆ブローゲーム。単一HTMLファイルのみで動作し、外部依存はありません。

### 遊び方

`hit-and-blow.html` をブラウザで開くだけです。

1. `> INITIATE HACK` でゲーム開始
2. 0〜9の中から重複しない3桁の数字を選んで `> EXECUTE`
3. 位置・数字とも一致 = **HIT**、数字のみ一致 = **BLOW**
4. 10回以内に正解（HIT: 3）すればクリア

キーボード操作にも対応（数字キー / Backspace / Enter / Escape）。

### 演出

- Web Audio API によるリアルタイム合成の効果音・BGM
- 1桁ずつ判定が確定するドラムロール演出
- パーティクル・紙吹雪・画面フラッシュ・グリッチ演出
- マトリックス風の背景演出とスキャンラインを使ったサイバーテーマ
